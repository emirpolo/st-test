_get

  • _get(obj: object, path: string, def?: any): any
    • Defined in utils.ts:14

    Gets the value at path of obj. If the resolved value is undefined, the def is returned in its place.

    example
    const object = { 'a': [{ 'b': { 'c': 3 } }] };
_get(object, 'a[0].b.c'); // => 3
_get(object, 'a.b.c', 'default'); // => 'default'
